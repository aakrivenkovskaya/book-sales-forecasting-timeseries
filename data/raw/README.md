# Synthetic Retail Book Sales Data (NDA-Safe)

This folder contains **synthetic datasets** used in the  
"Retail Book Sales Forecasting System" project.

All files replicate the structure and statistical behaviour of  
industry-standard retail book sales panel data, while containing  
**no real client information**.

### **`weekly_sales_synthetic.csv`**
Synthetic weekly point-of-sale (POS) sales data for each ISBN.

**Columns include:**  
- ISBN  
- Title  
- Author  
- Interval (weekly code)  
- End Date  
- Volume (units sold)  
- Value  
- ASP (average selling price)  
- RRP  
- Binding  
- Imprint  
- Publisher Group  
- Product Class  
- Weekly_Sheet  

---

These datasets are intended solely for demonstration purposes  
and do not reflect any real market data.
